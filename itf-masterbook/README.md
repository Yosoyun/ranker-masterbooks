# ARC · The Inverse Circular Functions

**The Ranker's Masterbook — 100 original problems on the inverse trigonometric functions**, built for
the very top of JEE Advanced and the Mathematical Olympiad shortlist. Every problem is *short to state,
brutal to solve*, and worked **more than one way**. The companion volume to **LIMITS · The Edge of
Calculus** (`../limits-masterbook`), in the same manuscript design.

The ten chapters are organised by the instrument that powers the problem:

| # | Chapter | Instrument |
|---|---------|-----------|
| I | Principal Branches & the Range Trap | sin⁻¹ |
| II | Addition Formulas & ±π Corrections | ±π |
| III | Telescoping Arctangents | Σ tan⁻¹ |
| IV | Identities & Collapse | ≡ |
| V | Equations & Root-Counting | = |
| VI | Trig of Inverse Trig | △ |
| VII | Domain, Range & Graphs | f∘g |
| VIII | The Calculus of Inverse Trig | d/dx |
| IX | Machin & Olympiad Identities | π |
| X | The Grand Hybrids | ⊕ |

Every problem ships with:
- a clean LaTeX **statement** and **answer**,
- **The Trap** — the seductive wrong answer a strong student writes (usually a dropped branch,
  quadrant or domain), and why it fails,
- **2–3 distinct solution methods** (principal-value reduction, the difference formula, the
  right-triangle picture, Gaussian-integer arguments, graphing, calculus…),
- an **Insight** note generalising the idea.

## Run it

It's a fully static site — no build, no dependencies, no server logic. Either:

```bash
# from this folder
python3 -m http.server 8760
# then open http://localhost:8760
```

…or just double-click `index.html` (everything works from `file://` too; KaTeX and the fonts load from
a CDN, so you need an internet connection the first time).

## Features

- **Manuscript UI** — Fraunces + Inter + KaTeX, an ivory "paper" theme and a "midnight" dark theme
  (toggle top-right or press `t`). Viridian accent (the Limits book is gold).
- **Chapter navigation** with per-chapter progress bars.
- **Filter** by difficulty (3 / 4 / 5), unsolved, or ★ bookmarked.
- **Search** across statements, titles and tags (press `/`).
- **Progress tracking** — mark problems solved, bookmark favourites; saved in `localStorage`.
- **Keyboard**: `/` search · `t` theme · `j` / `k` next / previous problem.
- **Print-friendly** — `Cmd/Ctrl+P` reveals every solution for a clean PDF.

## How the problems were made

The 100 problems were generated by an elite problem-setter fleet (one setter per chapter) and then
**adversarially verified**: a separate skeptical agent re-checked every claimed value, identity, root,
derivative and series **numerically in Python** (`mpmath`/`sympy`) — sampling across multiple branches
and quadrants, the exact place inverse-trig problems break — before any problem was admitted. Wrong
answers were recomputed and the solutions repaired.

## Files

| File | Role |
|------|------|
| `index.html` | Markup + CDN links |
| `styles.css` | The design system (ivory / midnight themes) |
| `app.js` | Routing, rendering, KaTeX, filters, progress |
| `problems.js` | The 100-problem dataset (`window.PROBLEMS`) |

To add or edit a problem, append an object to `window.PROBLEMS` in `problems.js` following the existing
shape (`theme`, `themeLabel`, `title`, `difficulty`, `task`, `tags`, `statement`, `answer`, `trap`,
`solutions[]`, `remark`). Statements and answers are **raw LaTeX**; solution steps use `$…$` / `$$…$$`.
