# C⁰ · The Continuous & the Broken

**The Ranker's Masterbook — original, olympiad/JEE-Advanced problems on continuity.** The fifth volume
in the masterbook series (LIMITS, ARC, TRIG, MATRIX), same engine, **crimson** theme. Every problem is
*short to state, deep to settle*, and worked **several ways**.

The ten chapters are organised by the instrument that powers the problem:

| # | Chapter | Instrument |
|---|---------|-----------|
| I | The ε–δ Definition | ε,δ |
| II | Classifying Discontinuities | ↯ |
| III | Continuous Functional Equations | f(x+y) |
| IV | The Intermediate Value Theorem | IVT |
| V | Pathological Functions | 1_ℚ |
| VI | Sequential & Composite Continuity | xₙ→ |
| VII | Parametric Continuity | a,b |
| VIII | Boundedness & Extreme Values | max |
| IX | Monotonic & Inverse Functions | ↗ |
| X | The Grand Hybrids | ⊕ |

Each problem ships with a clean LaTeX **statement** and **answer**, **The Trap**, **2–3 distinct
solution methods**, and an **Insight** note. Adversarially verified in Python (`sympy`/`numpy`/`mpmath`)
before admission. Run with `python3 -m http.server 8800` (or open `index.html`). PDFs built via
`tools/build-pdf.mjs`.
