# MATRIX · The Determinant & Its Eigenworld

**The Ranker's Masterbook — original, olympiad/Putnam-level problems on matrices & determinants**,
for the very top of JEE Advanced and the Mathematical Olympiad shortlist. Every problem is *short to
state, structurally deep*, and worked **several ways**. The fourth volume in the masterbook series,
alongside LIMITS, ARC and TRIG (same engine, **violet** theme).

The ten chapters are organised by the instrument that powers the problem:

| # | Chapter | Instrument |
|---|---------|-----------|
| I | The Determinant as an Alternating Form | det |
| II | Characteristic Polynomial & Cayley–Hamilton | λI |
| III | Rank, Nullity & Systems | rk |
| IV | Special Determinants (Vandermonde, circulant, tridiagonal) | Δ |
| V | Block Matrices & the Schur Complement | ⊞ |
| VI | Trace Magic & Commutators | tr |
| VII | Determinant Inequalities (Hadamard, Gram) | ≤ |
| VIII | Combinatorial & Integer Matrices | 0\|1 |
| IX | Matrix Equations | A² |
| X | The Grand Hybrids | ⊕ |

Every problem ships with a clean LaTeX **statement** and **answer**, **The Trap** (the seductive wrong
move and why it fails), **2–3 genuinely distinct solution methods**, and an **Insight** note. Problems
are drawn (originally, never copied) from the rarest corners of competition linear algebra.

## Run it

Fully static — no build, no dependencies:

```bash
python3 -m http.server 8790
# open http://localhost:8790
```

…or double-click `index.html` (KaTeX + fonts load from a CDN).

## How the problems were made

Generated by an elite problem-setter fleet and then **adversarially verified**: a separate skeptical
agent re-checked every determinant, eigenvalue, rank claim, Cayley–Hamilton power, inequality and
matrix-equation solution **symbolically and numerically in Python** (`sympy`/`numpy`) before any
problem was admitted. Wrong answers were recomputed and the solutions repaired.

## Files

| File | Role |
|------|------|
| `index.html` | Markup + CDN links |
| `styles.css` | The design system (ivory / midnight, violet accent) |
| `app.js` | Routing, rendering, KaTeX, filters, progress |
| `problems.js` | The problem dataset (`window.PROBLEMS`) |
| `tools/build-pdf.mjs` | PDF builder (puppeteer-core + katex) |

To add or edit a problem, append an object to `window.PROBLEMS` (`theme`, `themeLabel`, `title`,
`difficulty`, `task`, `tags`, `statement`, `answer`, `trap`, `solutions[]`, `remark`). Statements and
answers are raw LaTeX (matrices via `\begin{vmatrix}…\end{vmatrix}` / `\begin{pmatrix}…\end{pmatrix}`);
solution steps use `$…$` / `$$…$$`.
