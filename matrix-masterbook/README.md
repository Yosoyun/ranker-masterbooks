# MATRIX · Determinants, Inverses & Linear Systems

**The Ranker's Masterbook — original problems on matrices & determinants, strictly within the JEE
Advanced syllabus**, for the very top of JEE Advanced. Every problem is *short to state, structurally
deep*, and worked **several ways**. Part of the masterbook series (same engine, **violet** theme).

The ten chapters are organised by the instrument that powers the problem — and every problem lives
inside the JEE Advanced syllabus (matrices of order ≤3, determinants of order ≤3, the adjoint and
inverse, elementary transformations and systems of linear equations). There are **no** eigenvalues,
characteristic polynomials, rank/nullity, trace identities, Cayley–Hamilton, block decompositions or
determinant inequalities.

| # | Chapter | Instrument |
|---|---------|-----------|
| I | Algebra of Matrices | AB |
| II | Special Matrices (symmetric, skew, idempotent, nilpotent, involutory, orthogonal) | A²=A |
| III | Transpose & the Symmetric–Skew Decomposition | Aᵀ |
| IV | Determinants: Properties & Evaluation | \|A\| |
| V | Determinants: Factorisation, Area & Geometry | △ |
| VI | Adjoint & Inverse | A⁻¹ |
| VII | Elementary Transformations | R↔ |
| VIII | Systems of Linear Equations (Cramer's rule, consistency) | AX=B |

| X | The Grand Hybrids | ⊕ |

Every problem ships with a clean LaTeX **statement** and **answer**, **The Trap** (the seductive wrong
move and why it fails), **2–3 genuinely distinct solution methods**, and an **Insight** note. All
problems are original (never copied).

## Run it

Fully static — no build, no dependencies:

```bash
python3 -m http.server 8790
# open http://localhost:8790
```

…or double-click `index.html` (KaTeX + fonts load from a CDN).

## How the problems were made

Generated by an elite problem-setter fleet and then **adversarially verified**: a separate skeptical
agent re-checked every determinant, adjoint, inverse, Cramer's-rule solution and matrix-equation
**symbolically and numerically in Python** (`sympy`/`numpy`) before any problem was admitted, and a
syllabus pass confirmed no problem reaches beyond JEE Advanced. Wrong answers were recomputed and the
solutions repaired.

## Files

| File | Role |
|------|------|
| `index.html` | Markup + CDN links |
| `styles.css` | The design system (ivory / midnight, violet accent) |
| `app.js` | Routing, rendering, KaTeX, filters, progress |
| `problems.js` | The problem dataset (`window.PROBLEMS`) |
| `tools/build-pdf.mjs` | PDF builder (puppeteer-core + katex) |

To add or edit a problem, append an object to `window.PROBLEMS` (`theme`, `themeLabel`, `title`,
`difficulty`, `task`, `tags`, `statement`, `answer`, `trap`, `solutions[]`, `remark`). Statements and
answers are raw LaTeX (matrices via `\begin{vmatrix}…\end{vmatrix}` / `\begin{pmatrix}…\end{pmatrix}`);
solution steps use `$…$` / `$$…$$`.
