# v⃗ · Vectors & the Geometry of Space

**The Ranker's Masterbook — original, olympiad/JEE-Advanced problems on vectors & 3D geometry.** The
fourteenth volume in the series, same engine, **sky** theme. Don't reach for coordinates — let the
vectors do the geometry. Each problem worked **several ways**.

The ten chapters are organised by the instrument that powers the problem:

| # | Chapter | | # | Chapter |
|---|---------|---|---|---------|
| I | The Dot Product, Angles & Projections | | VI | Lines in Space |
| II | The Cross Product, Areas & Perpendiculars | | VII | Planes, Angles & Pencils |
| III | Scalar & Vector Triple Products | | VIII | Distances & the Shortest Distance Between Lines |
| IV | Linear Combinations, Bases & Reciprocal Systems | | IX | Vector Equations of Loci & Spheres |
| V | Position Vectors, Section Formula & Centroids | | X | The Grand Hybrids |

Each problem ships with a clean LaTeX statement & answer, **The Trap**, **2–3 distinct solution
methods**, and an **Insight** note. Adversarially verified in Python (`sympy`/`numpy`). Run with
`python3 -m http.server 8800` (or open `index.html`). PDFs via `tools/build-pdf.mjs`. Built to the
top-10 JEE-Advanced quality bar from the first pass.
