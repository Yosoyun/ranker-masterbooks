# x² · Roots, Discriminant & Vieta

**The Ranker's Masterbook — original, olympiad/JEE-Advanced problems on the quadratic.** The seventh
volume in the series (LIMITS, ARC, TRIG, MATRIX, C⁰, f′), same engine, **emerald** theme. The humblest
equation, pushed to the olympiad edge — every problem worked **several ways**.

The ten chapters are organised by the instrument that powers the problem:

| # | Chapter | | # | Chapter |
|---|---------|---|---|---------|
| I | Vieta & Symmetric Functions of Roots | | VI | Maxima & Minima |
| II | The Discriminant & Nature of Roots | | VII | Reducible to a Quadratic |
| III | Location of Roots | | VIII | Parameters |
| IV | Common Roots | | IX | Quadratic Inequalities |
| V | Range via the Discriminant | | X | The Grand Hybrids |

Each problem ships with a clean LaTeX statement & answer, **The Trap**, **2–3 distinct solution
methods**, and an **Insight** note. Adversarially verified in Python (`sympy`/`numpy`/`mpmath`). Run
with `python3 -m http.server 8820` (or open `index.html`). PDFs built via `tools/build-pdf.mjs`.
