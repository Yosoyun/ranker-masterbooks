# f′ · The Derivative & Where It Fails

**The Ranker's Masterbook — original, olympiad/JEE-Advanced problems on differentiability.** The sixth
volume in the series (LIMITS, ARC, TRIG, MATRIX, C⁰), same engine, **amber** theme. Every problem is
worked **several ways**.

The ten chapters are organised by the instrument that powers the problem:

| # | Chapter | | # | Chapter |
|---|---------|---|---|---------|
| I | Derivatives from First Principles | | VI | Higher Derivatives & Smoothness |
| II | Points of Non-Differentiability | | VII | Tangents, Normals & Curves |
| III | Mean Value & Rolle | | VIII | Monotonicity & Extrema |
| IV | Darboux & the Derivative's IVP | | IX | Derivative-driven Limits |
| V | Differentiable Functional Equations | | X | The Grand Hybrids |

Each problem ships with a clean LaTeX statement & answer, **The Trap**, **2–3 distinct solution
methods**, and an **Insight** note. Adversarially verified in Python (`sympy`/`numpy`/`mpmath`). Run
with `python3 -m http.server 8810` (or open `index.html`). PDFs built via `tools/build-pdf.mjs`.
