**The Ranker's Masterbook — original, olympiad/JEE-Advanced problems on indefinite integration.** The
eighteenth volume in the series, same engine, **brown** theme. Don't hunt blindly — recognise the derivative
that is already there. Each problem worked **several ways**.

Each problem ships with a clean LaTeX statement & answer, **The Trap**, **2–3 distinct solution
methods**, and an **Insight** note. Adversarially verified in Python (`sympy` — every antiderivative
differentiated back). Run with `python3 -m http.server 8820` (or open `index.html`). PDFs via
`tools/build-pdf.mjs`. Built to the top-10 JEE-Advanced quality bar from the first pass.
