# dy/dx · Solving the Equations of Change

**The Ranker's Masterbook — original problems on differential equations, strictly within the JEE
Advanced syllabus (first-order, first-degree only).** Same engine, **lime** theme. Don't just guess the
integrating factor — see why it must be what it is. Each problem worked **several ways**.

The ten chapters are organised by the instrument that powers the problem:

| # | Chapter | | # | Chapter |
|---|---------|---|---|---------|
| I | Formation, Order & Degree | | VI | Linear Equations & the Integrating Factor |
| II | Variables Separable | | VII | Initial-Value Problems |
| III | Reducible to Variables Separable | | VIII | Geometric Differential Equations |
| IV | Homogeneous Equations | | IX | Growth, Decay, Cooling & Mixing |
| V | Reducible to Homogeneous | | X | The Grand Hybrids |

Each problem ships with a clean LaTeX statement & answer, **The Trap**, **2–3 distinct solution
methods**, and an **Insight** note. Adversarially verified in Python (`sympy`/`numpy`). Run with
`python3 -m http.server 8805` (or open `index.html`). PDFs via `tools/build-pdf.mjs`. Built to the
top-10 JEE-Advanced quality bar from the first pass.
